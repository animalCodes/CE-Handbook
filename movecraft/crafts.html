Craft types

PLEASE NOTE: as of 2022/01/18 this and the blocks page are pretty much redundant due to c_corp creating the craftdata page on the official website, I will still work on this page so that this handbook is as complete as it can be, but I recommend you go there for more detailed info.

If you want to build a spaceship, it can be hard to choose what specific craft type to go for. Hopefully, this page will help you decide.

Most craft require a certain percentage of themselves to be 'flight blocks' which I will refer to as Fblocks. And engine blocks (Eblocks) in order to function. Additionally, most craft types will restrict how much of a certain block you can have on it, such as chests or barrels. (Sblocks) For a block guide see Valid blocks. Below is a list of all craft types and their stats.

Spaceships

Spaceships are the main craft type of the Celestial Expanse as they are the only craft type able to travel between planets.

Starfighter

Max size: 1150
Min size: 200
Fblock %: 35
Eblock %: 2.05
Sblock %: 1.5
Fblock: Wool/Carpet

Starfighters are the smallest and lightest spaceship, good for moving between bases / light defence but not much else, they are the only spaceship incapable of using a rift drive, and so are a poor choice for interplanetary travel. When you first join starcivs you will be given a S.I.B for a 'Guppie', which is a starfighter.

Cargoshuttle

Max size: 1150
Min size: 200
Fblock %: 25
Eblock %: 2.05
Sblock %: 17.5
Fblock: Wool/Carpet

Cargoshuttles are pretty much identical to starfighters with one key advantage: they can use Rift drives! For more info on rift drives see Interplanetary travel but TL:DR they let you warp between the orbits of planets. Making cargoshuttles ideal for quickly moving between planets.
